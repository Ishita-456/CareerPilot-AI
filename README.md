# 📖 Overview

CareerPilot AI is an **Agentic AI-powered Placement Mentor** built using **IBM watsonx Orchestrate**.

The agent provides personalized career guidance by understanding a student's goals, analyzing their current skills, identifying skill gaps, recommending projects, preparing interview plans, and generating structured learning roadmaps.

Instead of generic career advice, CareerPilot AI acts as an intelligent mentor that adapts recommendations based on each student's profile.

---

# 🎯 Problem Statement

Students often struggle with placement preparation because:

- Learning resources are scattered across multiple platforms.
- Guidance is generic instead of personalized.
- Students don't know which skills they are missing.
- Preparing for interviews and projects can feel overwhelming.

CareerPilot AI addresses these challenges by providing **AI-driven, personalized career guidance**.

---

# 💡 Solution

CareerPilot AI functions as an AI career mentor that can:

- 📊 Analyze student profiles
- 🎯 Identify skill gaps
- 🗺 Generate personalized learning roadmaps
- 💻 Recommend portfolio projects
- 🎤 Prepare students for interviews
- 📈 Estimate placement readiness
- 📅 Create weekly Mission Cards

## 📊 Skill Gap Analysis

- Recommend missing technologies
